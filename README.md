# AI-Financial-Advisor
Description
Python Environment Setup: Developed in Python 3.9 or above, leveraging its ecosystem for AI, data analytics, and visualization. Create and activate a virtual environment (e.g., ai_financial_advisor) for dependency management.

Official Website: https://www.python.org/downloads/

2. Streamlit Installation and Configuration: Streamlit powers the interactive dashboard for  financial data input, visualization, and AI-driven insights. 

Documentation: https://docs.streamlit.io/library/get-started/installation

Tutorial: https://www.youtube.com/watch?v=JwSS70SZdyM

3. Google Gemini API Integration: Integrates Google Gemini 2.0 Flash for personalized, AI powered financial recommendations. Create an account and generate an API key via Google. Resources: 

• Gemini SDK (PyPI): https://pypi.org/project/google-generativeai/

• Gemini Documentation: https://cloud.google.com/vertex-ai/generative-ai/docs

• Google AI Studio: https://aistudio.google.com/

4. Environment & Security Configuration (dotenv): Uses python-dotenv to securely store and  load environment variables such as the Gemini API key. 

Reference: https://pypi.org/project/python-dotenv/ 

5. Dependency Installation 

All dependencies are listed in requirements.txt. 

Key Libraries: 

• Streamlit – Interactive UI framework 

• Google Generative AI – Gemini model integration 

• Python-dotenv – Secure configuration handling 

• Pandas & NumPy – Data computation and analysis 

• Matplotlib & Seaborn – Visualization and analytics
